# 🤖 AI Interviewer

An AI Interviewer built using **HTML, CSS, JavaScript, FastAPI, and Ollama**.  
The application generates Python interview questions using AI and allows users to practice technical interviews.


<img width="1027" height="781" alt="image" src="https://github.com/user-attachments/assets/ee160ee1-222f-403e-9bc8-0ab59c671418" />


## 🚀 Features

- AI-generated Python interview questions
- Ollama (Llama 3.2) integration
- FastAPI backend
- Clean and responsive UI
- Answer input section
- Progress bar

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- Python
- FastAPI
- Ollama (Llama 3.2)

## ▶️ Run Project

```bash
pip install fastapi uvicorn ollama

cd backend
python -m uvicorn main:app --reload
```

Open `frontend/index.html` using **Live Server**.
